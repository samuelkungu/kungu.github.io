PRIVACY

The personal information you provide is used for delivery of the e-letters you have opted in to receive (e.g. your name, address, and/or e-mail address).

Data Analysis

This is pretty straight forward. The only data I’m collecting are analytics to track page views, and IP addresses via the default server logs.

For analytics, I’m using Plausible in place of Google Analytics. Plausible is a privacy-first analytics library that purposely doesn’t collect any personal data from users.

If you send me a message via the contact form, then your email,name and message will be stored in the Craft 3 database as well as being emailed to me (I used Fastmail.fm).
